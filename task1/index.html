Используется команда ping, с помощью которой найден IP-адресс сервера kubsu.ru: 185.13.84.92
nslookup — это команда, которая запрашивает DNS-сервер для получения IP-адреса по доменному имени (или наоборот), в данном случае команда возвращает А-записи kubsu.ru и kubsu-dev.ru: для kubsu.ru это 185.13.84.92, для kubsu-dev.ru - 212.192.134.135. (А-запись связывает доменное имя с IPv4-адресом)
